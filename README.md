# 🧑‍💻 CRUD de Usuários em Python

Sistema CRUD desenvolvido em Python com foco em lógica de programação, modularização e manipulação de arquivos.

## 📌 Funcionalidades

- Cadastro de usuários
- Listagem de usuários cadastrados
- Remoção de usuários
- Persistência de dados em arquivo `.txt`
- Tratamento de erros e validação de entradas
- Interface interativa via terminal

## 🧠 Conceitos praticados

- CRUD básico
- Modularização
- Funções
- Manipulação de arquivos
- Estruturas condicionais
- Estruturas de repetição
- Tratamento de exceções
- Validação de dados

## ⚙️ Tecnologias utilizadas

- Python 3
